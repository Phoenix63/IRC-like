# IRC-like
Chat IRC-like

##1. SERVER
###1.1 Install redis

First you need to install Redis :
https://github.com/rgl/redis/downloads
Add redis-server to environment variables.

Launch redis with the command

```
    redis-server
```

###1.2 Install dependencies
```
>>>>>>> origin/gbst3
    npm install
```

###1.3 Run the app
####1.3.1 As dev env
```
    npm run dev
```
####1.3.2 As prod env
```
    npm run start
```
