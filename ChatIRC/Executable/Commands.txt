!!! All channels names must start with # !!!
!!! All commands must br written in the "Debug" tab !!!


/nick <New nickname>  -- Setup a nickname
/user <username> 0 * : <real name> -- Auth on server, cant do aynthing before(guest only)
/join <channel>  -- Join a server
/names -- Users connected under every channels
/list  -- List every channels and their topics
/part <channel> --Leave the channel
/quit -- Leave the server
/who <channel> -- List user identity for every client in the channel
/msg <user> :<message> -- Send a private message to a user
<message> -- Send message to current channel
